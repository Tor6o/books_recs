Простой книжный агент, подбирающий книги для чтения на основании жанра и настроения с интерактивным режимом. В будущем планируется интеграция с Telegram, в формате чат-бота и добавление csv-файла с базой книг для создания большего разннобразия. В будущем возможно добавление новых форматов, типа квизов и викторин для вовлечения аудитории.
